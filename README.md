# 📊 Sales Performance Dashboard (Power BI)

This project is part of my Data Analytics Internship at Navyan (Project 1).

The goal of this project is to analyze sales data and build an interactive dashboard to extract meaningful business insights.

---

## 🚀 Project Overview

This dashboard provides a comprehensive view of sales performance across different regions, categories, and customer segments.

---

## 🔍 Key Insights

- West region contributes the highest share of overall sales  
- Consumer segment dominates revenue generation  
- Technology category leads in sales contribution  
- Sales show strong growth trend in the last quarter  

---

## 🛠 Tools & Technologies

- Power BI  
- Excel / CSV Dataset  
- Data Cleaning & Visualization  

---

## 📈 Dashboard Features

- KPI Metrics (Total Sales, Total Orders, Avg Order Value)  
- Sales by Region, Category, Segment  
- Monthly Sales Trend Analysis  
- Product & Sub-category Analysis  
- Interactive Filters (Region, Category, Segment, Ship Mode)
